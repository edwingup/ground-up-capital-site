# Portfolio v2 (Style Upgrade)

This patch/zip adds a premium, image-forward portfolio design scoped to portfolio pages only.

## Install (overview)
- Adds: `portfolio.v2.html`
- Adds: `portfolio/` (3 asset pages)
- Adds: `assets/css/portfolio-v2.css`
- Adds: `assets/js/portfolio-lightbox.js`
- Adds: `assets/img/portfolio/...` placeholders

After applying:
- Review locally, then swap in:
  - `mv portfolio.v2.html portfolio.html`

Image replacement:
- Replace `hero.jpg` and `gallery-01.jpg`..`gallery-12.jpg` in each asset folder.
